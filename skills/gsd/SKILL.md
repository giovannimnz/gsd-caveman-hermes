  # GSD Hermes

  Spec-driven development system for Hermes Agent.

  ## Quick Commands

  | Command | Description |
  |---------|-------------|
  | `/gsd-do` | Dispatcher - describe what you want |
  | `/gsd-new-project` | Start new project |
  | `/gsd-plan-phase N` | Plan phase N |
  | `/gsd-execute-phase N` | Execute phase N |
  | `/gsd-add-phase` | Add new phase |
  | `/gsd-debug` | Debug issue |
  | `/gsd-spike` | Experiment/validate |
  | `/gsd-sketch` | UI mockup |
  | `/gsd-research-phase` | Domain research |
  | `/gsd-verify-work` | Verify quality |
  | `/gsd-progress` | Check status |
  | `/gsd-help` | Full help |

  ## Happy Path

  1. `/gsd-new-project`
  2. `/gsd-plan-phase 1`
  3. `/gsd-execute-phase 1`
  4. `/gsd-validate-phase 1`
  5. Repeat for next phases...
